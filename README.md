# Sketch Pad
## [Live Preview](https://jclifford4.github.io/sketch-pad/)

### Description
Resizable drawing pad for the web

### Functionality
* Dynamically resizable drawing area
* Color selection
* Erasing capability
* Clear grid entirely

### Code Base
* JavaScript
* HTML
* CSS
